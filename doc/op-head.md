## head

Truncate dataset to its first rows.

### Usage

    head [[-n] COUNT]

    COUNT
        Number of rows to truncate to. If this option is
        omitted, it defaults to 10.

Note we allow you to prefix `COUNT` with `-n` to ensure compatability with standard unix `head`.
